# Compliance Checklist

## 1. FERPA

*   [ ] **Data privacy policy:**
    *   [ ] The data privacy policy must be compliant with FERPA.
    *   [ ] The data privacy policy must be made available to all users.
*   [ ] **Data encryption:**
    *   [ ] All student data must be encrypted in transit and at rest.
*   [_] **Data access controls:**
    *   [ ] Access to student data must be restricted to authorized personnel.
*   [ ] **Data retention policy:**
    *   [ ] The data retention policy must be compliant with FERPA.
    *   [ ] The data retention policy must be made available to all users.
*   [ ] **Data breach response plan:**
    *   [ ] The data breach response plan must be compliant with FERPA.
    *   [ ] The data breach response plan must be tested on a regular basis.

## 2. Other Regulations

*   [ ] **COPPA:**
    *   [ ] The application must be compliant with COPPA if it is directed to children under 13.
*   [ ] **GDPR:**
    *   [ ] The application must be compliant with GDPR if it is used by individuals in the European Union.
*   [ ] **CCPA:**
    *   [ ] The application must be compliant with CCPA if it is used by individuals in California.
